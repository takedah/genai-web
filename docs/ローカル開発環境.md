# ローカル開発環境

## 前提条件

[デプロイ手順](./デプロイ手順.md)に従って環境が構築済みであること。

## 実行

```
sh scripts/run.sh 環境名（例: -dev）
```

成功した場合、以下のメッセージがコンソールに出力されます。

```
➜  Local:   http://localhost:5173/
➜  Network: use --host to expose
➜  press h + enter to show help
```

http://localhost:5173/ にアクセスし、動作確認をしてください。

## 変更が反映されるか確認

試しに `packages/web/src/features` 以下のコードのいずれかを変更してください。  
http://localhost:5173/ にアクセスし、変更が反映されているか確認してください。

## 注意点

ローカル開発環境では `packages/web/` 以下の変更は即時反映されますが、`packages/cdk/` 以下の変更は再度デプロイが必要になります。
