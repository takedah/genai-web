# 事前準備

デプロイ手順のドキュメントは、閉域ネットワークへデプロイできるようにするため、デジタル庁 [genai-web](https://github.com/digital-go-jp/genai-web) を元に一部改変しています。ライセンスは CC BY 4.0 のもとで提供されています。

デプロイを行う前に各種ツールのインストールと設定を行う必要があります。

## Node.js のインストール

以下のページからインストールしてください。
（`.node-version` で指定されているバージョンをインストールしてください。）
https://nodejs.org/ja/download

[mise](https://mise.jdx.dev/getting-started.html) を使用している場合は、以下のコマンドでインストールできます。

```bash
mise trust
mise install
```

### Node.js のバージョン確認

```bash
# .node-version もしくは mise.toml に記載されているバージョンが表示されれば問題ありません
node -v
```

## AWS CLI のインストール

以下のドキュメントを参考にインストールしてください。
https://docs.aws.amazon.com/ja_jp/cli/latest/userguide/getting-started-install.html

インストール後、ご自身の AWS 環境に合わせて認証情報を設定してください。

## AWS CDK CLI のインストール

以下のドキュメントを参考にインストールしてください。
https://docs.aws.amazon.com/ja_jp/cdk/v2/guide/getting-started.html

## 依存関係のインストール

リポジトリのルートディレクトリで以下を実行し、依存ライブラリをインストールしてください。

```bash
npm ci
```

## AWS CDK の Bootstrap

初めて CDK を使用する場合、またはこの AWS アカウントとリージョンの組み合わせで初めて CDK をデプロイする場合は、Bootstrap が必要です。

```bash
npm -w packages/cdk run cdk -- bootstrap
```

## jq のインストール

以下のページからインストールしてください。
https://jqlang.org/download/

## Docker のインストールとサービスの起動

ECS で起動する Docker イメージをビルドするために Docker が必要です。Docker をインストールし、CDK Deploy の前にサービスを起動してください。

https://docs.docker.com/get-started/get-docker/
