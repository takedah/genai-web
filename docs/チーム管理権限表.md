# チーム管理権限表

## ロール定義

### Cognito グループ

| グループ名         | 説明                 | 付与タイミング                                            |
| ------------------ | -------------------- | --------------------------------------------------------- |
| `SystemAdminGroup` | システム全体の管理者 | 管理者が手動で付与                                        |
| `TeamAdminGroup`   | チームの管理者       | チーム作成時またはメンバー追加時に `isAdmin: true` で付与 |
| `UserGroup`        | 一般ユーザー         | サインアップ時に自動付与                                  |

### TeamUser レコードの `isAdmin` フラグ

Cognito グループに加え、DynamoDB のチーム-ユーザーレコードにも `isAdmin` フラグがあり、チーム単位の管理権限を制御します。

- `isAdmin: true` → 該当チームの管理者
- `isAdmin: false` → 該当チームの一般メンバー

### 有効な権限の組み合わせ

| ロール名       | Cognito グループ   | TeamUser `isAdmin` | 説明                                       |
| -------------- | ------------------ | ------------------ | ------------------------------------------ |
| システム管理者 | `SystemAdminGroup` | -                  | すべてのチーム・アプリに対して全操作が可能 |
| チーム管理者   | `TeamAdminGroup`   | `true`             | 所属チームの管理操作が可能                 |
| 一般メンバー   | `UserGroup`        | `false`            | 所属チームのアプリ閲覧・実行のみ可能       |

## 共通チーム（TEAM_ID: `00000000-0000-0000-0000-000000000000`）

`TEAM_ID: 00000000-0000-0000-0000-000000000000` に登録されたアプリは、チームメンバーの権限に関係なく **全認証済みユーザー** がアクセス・実行可能です。

## API 権限一覧

### チーム管理

| 操作                     | API          | SystemAdmin | TeamAdmin（自チーム） | 一般メンバー | 権限チェック関数                 |
| ------------------------ | ------------ | :---------: | :-------------------: | :----------: | -------------------------------- |
| チーム一覧取得           | `listTeams`  | ✅ 全チーム |   ✅ 管理チームのみ   |      ❌      | `isSystemAdmin` 分岐             |
| チーム詳細取得           | `getTeam`    |     ✅      |          ✅           |      ❌      | `requireTeamAdminOrSystemAdmin`  |
| チーム詳細取得（管理用） | `getRawTeam` |     ✅      |          ✅           |      ❌      | `isTeamAdmin \|\| isSystemAdmin` |
| チーム作成               | `createTeam` |     ✅      |          ❌           |      ❌      | `requireSystemAdmin`             |
| チーム更新               | `updateTeam` |     ✅      |          ✅           |      ❌      | `requireTeamAdminOrSystemAdmin`  |
| チーム削除               | `deleteTeam` |     ✅      |          ❌           |      ❌      | `requireSystemAdmin`             |

### メンバー管理

| 操作                     | API              | SystemAdmin | TeamAdmin（自チーム） | 一般メンバー | 権限チェック関数                |
| ------------------------ | ---------------- | :---------: | :-------------------: | :----------: | ------------------------------- |
| メンバー一覧取得         | `listTeamUsers`  |     ✅      |          ✅           |      ❌      | `requireTeamAdminOrSystemAdmin` |
| メンバー詳細取得         | `getTeamUser`    |     ✅      |          ✅           |      ❌      | `requireTeamAdminOrSystemAdmin` |
| メンバー追加             | `createTeamUser` |     ✅      |          ✅           |      ❌      | `requireTeamAdminOrSystemAdmin` |
| メンバー更新（権限変更） | `updateTeamUser` |     ✅      |          ✅           |      ❌      | `requireTeamAdminOrSystemAdmin` |
| メンバー削除             | `deleteTeamUser` |     ✅      |          ✅           |      ❌      | `requireTeamAdminOrSystemAdmin` |

> **制約**: チーム管理者が 0 人になる操作（最後の管理者の削除・権限変更）はエラーとなります。

### AIアプリ管理

| 操作                           | API              | SystemAdmin | TeamAdmin（自チーム） | 一般メンバー（自チーム） |  未所属ユーザー   | 権限チェック関数                                    |
| ------------------------------ | ---------------- | :---------: | :-------------------: | :----------------------: | :---------------: | --------------------------------------------------- |
| アプリ一覧取得                 | `listExApps`     | ✅ 全チーム |     ✅ 所属チーム     |      ✅ 所属チーム       | ✅ 共通チームのみ | 所属チーム + 共通チーム                             |
| アプリ詳細取得                 | `getExApp`       |     ✅      |          ✅           |            ✅            | ✅ 共通チームのみ | `isSystemAdmin \|\| isTeamUser \|\| COMMON_TEAM_ID` |
| アプリ詳細取得（管理用）       | `getRawTeamApp`  |     ✅      |          ✅           |            ❌            |        ❌         | `isTeamAdmin \|\| isSystemAdmin`                    |
| アプリ一覧取得（チーム管理用） | `listTeamExApps` |     ✅      |          ✅           |            ❌            |        ❌         | `requireTeamAdminOrSystemAdmin`                     |
| アプリ作成                     | `createExApp`    |     ✅      |          ✅           |            ❌            |        ❌         | `requireTeamAdminOrSystemAdmin`                     |
| アプリ更新                     | `updateExApp`    |     ✅      |          ✅           |            ❌            |        ❌         | `requireTeamAdminOrSystemAdmin`                     |
| アプリ削除                     | `deleteExApp`    |     ✅      |          ✅           |            ❌            |        ❌         | `requireTeamAdminOrSystemAdmin`                     |
| アプリコピー                   | `copyExApp`      |     ✅      |          ✅           |            ❌            |        ❌         | `requireTeamAdminOrSystemAdmin`                     |
| アプリ実行                     | `invokeExApp`    |     ✅      |          ✅           |            ✅            | ✅ 共通チームのみ | `isSystemAdmin \|\| isTeamUser \|\| COMMON_TEAM_ID` |

> **備考**: AIアプリのステータスが `draft` であっても、チームメンバーは URL（`teamId` と `exAppId`）を知っていれば詳細の取得および実行が可能です。
